# React Context + Routing Lab (ITM301)

Даалгаврын жишээ хэрэгжилт.

## Суурилуулалт
```bash
npm install
npm run dev
```

## Замчлал
- `/` — Хэрэглэгчдийн жагсаалт
- `/:uid/places` — Сонгосон хэрэглэгчийн газрууд
- `/authenticate` — Нэвтрэх/Бүртгүүлэх
- `/places/new` — (Нэвтэрсэн үед) Шинэ газар нэмэх
- `/places/:pid` — Газрын дэлгэрэнгүй (устгах боломжтой)
```

Локал өгөгдөл нь LocalStorage-д хадгалагдана.
